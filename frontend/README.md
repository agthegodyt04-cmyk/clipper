# Clipper Frontend

React + Vite UI for:

1. Project setup and AI copy/image generation
2. Brush mask editor and inpaint job submission
3. Storyboard and T2V job submission with video preview

Run:

```powershell
cd d:\clipper\clipper\frontend
npm install
npm run dev
```

Backend should run at `http://127.0.0.1:8000` unless `VITE_API_BASE_URL` is set.
